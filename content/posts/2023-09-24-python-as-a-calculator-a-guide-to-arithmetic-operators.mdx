### Python as a Calculator: A Guide to Arithmetic Operators

#### Python Programming

In this blog post, we’ll explore how Python’s interpreter can be used as a calculator for mathematical operations. Python’s interpreter can be used as a calculator to perform calculation.

```
2 + 2
```

```
4
```

To perform mathematical calculation, it’s essential to understand the inner working of **arithmetic operators** and their **priorities**.

### Arithmetic Operators in Python

Operators in Python are symbols used to operate on values and produce a result. Here are the basic arithmetic operators in Python, also known as **numerical** operators:

`+`, `-`, `*`, `/`, `//`, `%`, `**`

When an operator is combined with data, it forms an **expression**. *Expressions always evaluate down to a value*.

```
5 - 3
```

```
2
```

This is a simple expression, which goes down to a single value.

#### Exponentiation `**` Operator

The exponentiation operator in Python performs exponentiation.

```
2 ** 3
```

```
8
```

In this example above, the left digit `2` is called the **base**, and the right digit `3` is called the **exponent**. When both arguments are **integers**, the result is also an integer, if one argument in the expression is a **float**, the result will be a float.

```
2 ** 3.0
```

```
8.0
```

#### Multiplication `* Operator`

Multiplication works similarly to exponentiation. If both values are integers, the result will be an integer, or a float when one argument is a float.

```
print(2 * 3)  
print(4.5 * 2)
```

```
6  
9.0
```

#### Division `/ Operator`

In mathematics, division involves a **dividend** and a **divisor**. The result produced by the division operator is **always a float**.

```
print(5 / 2)  
print(4.2 / 2)
```

```
2.5  
2.1
```

#### Integer Division `// Operator`

Integer division always rounds down the result to the nearest integer.

```
7 // 2  # 7/2 = 3.5, the lesser part is 3
```

```
3
```

Compare these examples with the division operator to see the difference. Integer division rounds towards the **lesser integer part**, often referred to as **floor division**.

```
print(-6 // 4)  # -6/4 = -1.5, the lesser part is -2
```

```
-2
```

#### Modulo (Remainder) `% Operator`

The modulo operator (%) is used to find the remainder after the integer division.

```
14 % 4
```

```
2
```

If we divide `14 // 4`, the result is `3`. Multiplying `3` by `4` gives us `12`, and the remainder is `14 - 12 = 2`.

#### Addition `(+) Operator`

The addition operator functions as expected, following mathematical standards.

#### Subtraction (`-) Operator`

Similarly, the subtraction operator behaves like it does in mathematics.

### Operators and Their Priorities

In expressions, operators have different priorities and precedence as below:

1. Exponent `(**)`
2. Unary `(+, -)`
3. Multiplication `(*)`, Division `(/)`, Integer Division `(//)`, Modulo `(%)`
4. Addition `(+)`, Subtraction `(-)`

**Note:** Sub-expressions enclosed in parentheses are always evaluated first.

> Unary operators work with only one operand `(e.g: -1 or +1)`, while binary operators work with two operands `(e.g: 1 + 2 or 2 - 1)`.

#### Binding of Operators

Binding refers to the direction in which operators are evaluated, either from left to right or from right to left.

```
2 / 4 * 2
```

```
1.0
```

In the above code, when their priorities are the same, operations proceed from left to right, except for the exponentiation operator, which is right-to-left binding.
